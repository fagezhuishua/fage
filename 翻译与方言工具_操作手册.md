# 翻译与方言工具（Translation-tool.py）操作手册

> 版本：v2.1  
> 技术栈：Python 3 + customtkinter + deep-translator + edge-tts + pygame

---

## 一、功能概述

`Translation-tool.py` 是一款基于 modern GUI 的翻译与方言语音工具，集成了**国际语言翻译**和**汉语方言/地区语音转换**两大核心功能，并支持文本转语音（TTS）播放。

### 核心功能

| 功能 | 说明 |
|------|------|
| 国际翻译 | 支持 14 种国际语言互译，调用 Google 翻译接口 |
| 方言翻译 | 支持 9 种汉语方言/地区发音互译，基于标准中文识别 |
| 语音播放 | 使用 Edge TTS 将原文或译文合成语音并播放 |
| 深色/浅色主题 | 支持界面主题切换 |
| 多语言界面 | 支持中文、英文两种界面语言 |
| 文本操作 | 支持清空、复制、交换源/目标语言 |

## 二、运行环境

### 2.1 必需依赖

```bash
pip install customtkinter pygame
```

### 2.2 可选依赖

| 依赖 | 用途 | 缺少时的影响 |
|------|------|-------------|
| `deep-translator` | Google 翻译 API 调用 | 翻译功能不可用 |
| `edge-tts` | 微软 Edge 在线 TTS | 语音播放功能不可用 |
| `Pillow` | 背景渐变图片生成 | 无背景图，但功能正常 |

推荐全部安装：

```bash
pip install deep-translator edge-tts Pillow
```

### 2.3 运行方式

```bash
python Translation-tool.py
```

> 注：该程序为 GUI 应用，**没有命令行参数**。

### 4.2 翻译引擎模块（TranslationEngine）

```python
class TranslationEngine:
    @staticmethod
    def translate(text, source_code, target_code)
```

- 使用 `deep-translator` 库调用 `GoogleTranslator`。
- 在后台线程中执行，避免阻塞 GUI。
- 翻译结果通过 `self.after()` 回写到主线程更新界面。

### 4.3 TTS 语音引擎模块（TTSEngine）

```python
class TTSEngine:
    @classmethod
    def init(cls)
    @classmethod
    def generate(cls, text, voice)
    @classmethod
    def play(cls, audio_path, on_finish=None)
    @classmethod
    def stop(cls)
```

- 使用 `edge-tts` 调用微软 Edge 在线语音合成服务。
- 生成 `.mp3` 临时文件保存到系统临时目录。
- 使用 `pygame.mixer` 播放音频。
- 播放完成后通过回调函数更新状态栏。

### 4.4 设置持久化模块

```python
SETTINGS_FILE = "translation_settings.json"
DEFAULT_SETTINGS = {"language": "zh_CN", "theme": "Light"}
```

- 程序启动时加载 `translation_settings.json`。
- 界面语言和主题切换后自动保存。
- 语言切换需重启程序生效。

### 4.5 多语言界面模块（TEXTS）

支持两种界面语言：

- `zh_CN`：中文界面
- `en`：英文界面

通过 `_t(key)` 方法根据当前语言获取对应文本。

### 4.6 背景渲染模块

- 使用 PIL 生成渐变背景图。
- 窗口大小变化时延迟 300ms 重新渲染，避免频繁刷新。
- 若未安装 Pillow，背景不显示，但不影响功能。

## 五、操作流程

### 5.1 国际翻译

1. 切换到 **「国际翻译」** 标签页。
2. 在「源语言」下拉框选择输入文本的语言。
3. 在「目标语言」下拉框选择要翻译成的语言。
4. 在左侧原文输入框输入或粘贴文本。
5. 点击 **「开始翻译」** 按钮。
6. 右侧译文输出框显示翻译结果。
7. 可点击 **「播放原文」** 或 **「播放译文」** 收听语音。
8. 点击 **「复制结果」** 可将译文复制到剪贴板。
9. 点击 **「清空」** 可清空原文和译文。

### 5.2 方言翻译

1. 切换到 **「方言翻译」** 标签页。
2. 在「源语言」下拉框选择源方言/发音（如普通话）。
3. 在「目标语言」下拉框选择目标方言/发音（如粤语）。
4. 在左侧原文输入框输入标准中文文本。
5. 点击 **「开始翻译」** 按钮。
6. 右侧输出框显示基于标准中文识别的结果（方言翻译主要为展示对应方言的 TTS 发音）。
7. 点击 **「播放译文」** 可收听目标方言/地区的语音。

### 5.3 交换语言

点击语言选择行中的 **「交换」** 按钮，可快速交换源语言和目标语言。

### 5.4 主题切换

在左侧边栏点击 **「深色模式」** 开关，可切换浅色 / 深色主题。

### 5.5 界面语言切换

在左侧边栏选择 **「界面语言」** 下拉框，切换后需重启程序生效。

## 六、注意事项

### 6.1 网络依赖

- **翻译功能** 依赖 Google 翻译接口，需要稳定的网络连接。
- **语音播放功能** 依赖微软 Edge TTS 在线服务，需要网络连接。
- 若网络异常，会提示「网络连接失败，请检查网络」或具体错误信息。

### 6.2 TTS 文本长度限制

- 当前 TTS 生成时只取文本前 100 个字符：`safe_text = text[:100]`。
- 长文本播放会被截断，如需完整朗读，请分段输入。

### 6.3 方言翻译说明

- 方言翻译并非真正的方言互译，而是基于标准中文识别的文本转换。
- 主要区别在于 **TTS 语音** 使用不同地区的发音模型。
- 状态栏已提示：「方言翻译基于标准中文识别，结果仅供参考」。

### 6.4 Office 无关

- 本工具不涉及 Office COM 调用，无需安装 Microsoft Office。

### 6.5 临时文件

- TTS 生成的 `.mp3` 文件保存在系统临时目录 `tempfile.gettempdir()/translation_tts` 下。
- 程序不会自动清理这些临时文件，建议定期手动清理。

### 6.6 语言设置生效规则

- 主题切换后即时生效并保存。
- 界面语言切换后，需要重启程序才能生效。

### 6.7 翻译结果准确性

- 机器翻译结果仅供参考，重要文档翻译后请人工校对。
- Google 翻译对专业术语、俚语、方言表达的翻译可能不够准确。

### 6.8 输入占位符

- 文本框默认显示占位提示文本（如「在此输入要翻译的文本...」）。
- 点击文本框即可输入，程序会自动处理占位符。

### 6.9 并发限制

- 翻译和 TTS 播放均在后台线程执行，避免界面卡死。
- 翻译过程中「开始翻译」按钮会禁用，完成后恢复。

## 七、常见问题（FAQ）

**Q1：翻译按钮点击后没有反应？**

请检查：
1. 是否已安装 `deep-translator`：
   ```bash
   pip install deep-translator
   ```
2. 网络是否通畅，能否访问 Google 翻译服务。
3. 原文输入框是否为空或仅为占位符。

**Q2：语音播放按钮点击后没有声音？**

请检查：
1. 是否已安装 `edge-tts` 和 `pygame`：
   ```bash
   pip install edge-tts pygame
   ```
2. 网络是否通畅，能否访问微软 Edge TTS 服务。
3. 系统音量是否正常。

**Q3：粤语为什么在国际翻译中找不到？**

粤语属于汉语方言，已从国际语言列表中移除，统一归类到「方言翻译」标签页中。

**Q4：界面语言切换后为什么还是原来的语言？**

界面语言切换后需要重启程序才能生效，程序会在切换时弹出提示。

**Q5：为什么有些方言播放出来像普通话？**

部分方言（如闽南话、江西话、天津话）目前没有独立的 Edge TTS 语音模型，程序使用普通话语音作为回退，因此发音接近普通话。
